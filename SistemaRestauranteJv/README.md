# Sistema de Restaurante em Java 🍽️

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO).

O sistema permite criar restaurantes, clientes, pratos e pedidos utilizando atributos, construtores, getters, setters e métodos personalizados para a lógica do sistema, como cálculo total do pedido, histórico de pedidos e controle de status.

## Conceitos utilizados
- POO
- Encapsulamento
- ArrayList
- Construtores
- Getters e Setters
- Relacionamento entre classes
